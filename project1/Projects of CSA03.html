Project 1 of CSA03


There is no writing exam for CSA03 (Nature Inspired Design). However, a student is required to join in a team that might have 4 members at most. Each team is required to finish two projects. "Project 1" assigned today is to apply evolutionary algorithms on optimization tasks. "Project 2" assigned next time is to apply evolutionary algorithms on learning tasks. You can choose "Project 1" given as follows. However, it is just an example for Project 1. In Project 1, each team can freely choose any other optimization tasks by applying a learned evolutionary algorithm.

1. Project 1

Compare the performance between  Evolutionary Programming (EP) and Fast Evolutionary Programming (FEP) on at least 2 multimodal functions.

You could implement EP and FEP based on the Lectures 2 and 3. The details of the implementations of  EP and FEP can be found in the paper:

X. Yao, Y. Liu and G. Lin, ``Evolutionary programming made faster,'' IEEE Transactions on Evolutionary Computation, 3(2):82-102, July 1999. Over 4268 citations at Google Scholar

You could also find the test multimodal functions in the given paper. For the test functions, you could either use the functions given in the paper or choose any other functions not included in the given paper.

The purpose of this project is to let you know how important the search step sizes are in evolutionary algorithms, and how to adjust the search step sizes during the evolutionary process.
                                                           

2. Project Report

Two weeks are given for doing the project. Submit your team project report in PDF by emails (subject as "Project 1 of CSA03") before the deadline. The report should include the following parts:
  1. descriptions of the algorithms used.
  2. descriptions of the test problems used;
  3. descriptions of the simulation results;
  4. the code used to get the results.
